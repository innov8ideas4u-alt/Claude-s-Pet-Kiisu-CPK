The goal of this wiki is to help document the [Flipper Zero](https://flipperzero.one).  The code section of this repository has various samples and README.md files that help explain some of the key concepts of the sample.  This wiki is where we can document the latest understanding of the Flipper Zero and technologies related to it.  There are also various YouTube channels with technical content on the Flipper Zero.  Many of the tutorials in this repository have a YouTube video in [@MrDerekJamison](https://youtube.com/@MrDerekJamison).
