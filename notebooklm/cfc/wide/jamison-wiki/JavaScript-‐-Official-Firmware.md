# Overview
JavaScript on the Official firmware is still under development.

https://developer.flipper.net/flipperzero/doxygen/js.html is current docs.
- DOCBUG: [to_string(x)](https://developer.flipper.net/flipperzero/doxygen/js_builtin.html) is now x.toString()

Example scripts can be found in the Official firmware under the [js_app/examples](https://github.com/flipperdevices/flipperzero-firmware/tree/dev/applications/system/js_app/examples/apps/Scripts) folder. You can also ask people in the [Flipper Devices Discord](https://discord.gg/flipper) server.
